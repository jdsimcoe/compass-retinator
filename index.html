    • Compass retina sprites

    An easy way to generate retina ready sprites

    Project maintained by Restorando Hosted on GitHub Pages — Theme by mattgraham

    This Compass extension will allow to create retina ready sprites from the scratch. Juast drop your images on the right folders, type some Compass mumble jumble and you are rady to go!

    Installation 

    gem installl compass-retina-sprites

    How to use it?

    Fisrt thing to do is drop your images in the right folders. In this case I have to folders icons and icons-sprites inside my image folder.

    Next, we need to tell Compass where those image are. Add this line to variables.saas or somewhere else 

    $sprites:        sprite-map("icons/*.png")
$sprites_retina: sprite-map("icons-sprites/*.png")

    Now we need to init awesomeness, add this line to your project: 

    @import "compass-retina-sprites"

    Using the extension

    You can pass a bouch of parameters to obtain the right result, some valid options are: 

    // This will import plain.png
.plain
  +sprite(plain)

// This will import boat.png, boat_hover.png and boat_active.png
.boat
  +sprite(boat, $hover_state: true, $active_state: true)

// This will import console.png plus 40px of padding on the sides
.console
  +sprite(console, $spacing: 40px)

// This will import people.png and it will not set any dimension on this elemments
.people
  +sprite(people, $dimensions: False)
